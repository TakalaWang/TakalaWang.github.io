師大資工唯一沒修過就不配當資工系學生的課（沒，地圖炮開太兇了，其他老師也上得很好，能畢業的都是好學生）
課程內容超級超級扎實，無限的作業保證你會修的很開心，而且內容幾乎包括計概的內容，所以你可以不用去其他系浪費時間修計概。
修完課你的程式能力應該會有一個很大的進步（不進步也得進步），作業很多，題目很難，不是常規的演算法題目，都是實務型的題目，因此會花大量時間在打 code 上。
會有五次作業，期中考跟期末考，以及大量的 bonus 作業，並且基本上每年都會調分，所以看到作業太低，bonus 記得好好爭取。
反正，修就對了！
